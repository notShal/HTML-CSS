HTML5 - Tabelas

  • < table >Define a tabela
  • < th >Define cabeçalho da célula na tabela
  • < tr >Define a linha na tabela
  • < td >Define a célula na tabela
  • < caption >Define um título pra tabela
  • < colgrup >Especifica um grupo de uma ou mais colunas em uma tabela para formatação.
  • < col >Especifica propriedades de coluna para cada coluna dentro de um elemento <colgroup >
  • < thead >Agrupa o conteúdo do cabeçalho em uma tabela
  • < tbody >Agrupa o conteúdo do corpo em uma tabela
  • < tfoot >Agrupa o conteúdo do rodapé em uma tabela

A tag colgroup é útil para aplicar estilos a colunas inteiras, em vez de repetir os estilos para cada célula, para cada linha.
Observação: a tag colgroup deve ser filha de um elemento table, depois de qualquer elemento caption e antes de qualquer elemento thead, tbody, tfoot e tr.

Usando CSS em tabelas

Podemos definir o tamanho da tabela com width (largura) e height (altura).

Também devemos adcionar boradas para facilitar a vizualização das células das tabelas.
No exemplo abaixo usei border-collapse:collapse*, border e suas propiedades dentro de table e border dentro de td

Nota: Quando você aplica border-collapse: collapse; a uma tabela HTML, você está especificando que as bordas das células da tabela devem ser combinadas, ou seja, as bordas adjacentes se fundirão em uma única linha sólida. Isso cria uma aparência mais coesa e limpa para a tabela. Quando as bordas colapsam, elas compartilham os segmentos de borda comuns, o que resulta em uma aparência mais nítida e menos espaçamento entre as células da tabela.
Por outro lado, quando border-collapse não é definido ou definido como border-collapse: separate;, as bordas das células da tabela são renderizadas como linhas individuais separadas, o que pode levar a um espaçamento visível entre as células adjacentes.
